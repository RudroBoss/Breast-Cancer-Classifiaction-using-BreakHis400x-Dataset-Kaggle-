# Breast-Cancer-Classifiaction-using-BreakHis400x-Dataset-Kaggle-
CapsNet-based breast cancer classification on the BreaKHis 400X dataset with class balancing, heavy augmentation, weighted loss, CosineAnnealingLR &amp; early stopping. Achieved 99.49% best val acc and 98.18% test acc. Grad-CAM used for model interpretability and lesion-focused visualization.
